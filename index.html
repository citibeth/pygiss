Introduction

pyGISS is a general Python library. It is needed for a number of things, including GLINT2. It was developed by Robert Fischer (@citibob) at NASA GISS in New York City, and is licensed via the GPL v3.

pyGISS code may be downloaded at this GitHub repository. pyGISS documentation is found at this project's Wiki.

If you are interested in using pyGISS or joining as a committer, please email Robert Fischer robert.fischer@nasa.gov
